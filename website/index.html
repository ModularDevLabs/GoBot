Install Path

1. Download or build

go build -o modbot ./cmd/modbot

2. Run

MODBOT_TOKEN=your_token \
MODBOT_ADMIN_PASS=your_pass \
./modbot --db modbot.sqlite

3. Configure modules

Open http://127.0.0.1:8080
Select guild
Enable modules as needed

FAQ

Can this live in the same repo as the bot?

Yes. That is a common OSS pattern. Website and product versions stay aligned and release links stay accurate.

Do I need all modules enabled?

No. New guilds default to disabled modules. Enable only the feature set your community needs.

Can I use binaries instead of building?

Yes. Use GitHub Releases for Linux, macOS, and Windows artifacts.
